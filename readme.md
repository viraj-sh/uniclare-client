# Unofficial Uniclare Client

A local client for the **Uniclare App** built for college use.
This client provides additional functionalities and quality-of-life improvements over the regular Uniclare interface.
Anyone who wishes to use it may do so by following the instructions below.

---

## Installation & Setup

1. **Clone this repository**

   ```bash
   git clone https://github.com/viraj-sh/uniclare-client
   cd uniclare-client
   ```

2. **Update pip**

   ```bash
   python -m pip install --upgrade pip
   ```

3. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application**

   ```bash
   python app.py
   ```

   **For development mode:**

   ```bash
   uvicorn app:app --reload
   ```
